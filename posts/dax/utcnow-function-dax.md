# UTCNOW

Returns the current UTC date and time.
  
## Syntax  
  
```dax
UTCNOW()  
```
  
## Return value

A (**datetime)**.  
  
## Remarks  

The result of the UTCNOW function changes only when the formula is refreshed. It is not continuously updated. 
  
## Example

The following:
  
```dax
EVALUATE { FORMAT(UTCNOW(), "General Date") }
```

Returns:

|[Value]  |
|---------|
|2/2/2018 4:48:08 AM    |
