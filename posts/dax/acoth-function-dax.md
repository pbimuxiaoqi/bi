# ACOTH

Returns the inverse hyperbolic cotangent of a number.
  
## Syntax  
  
```js
ACOTH(number)  
```
  
### Parameters  
  
|Term|Definition|  
|--------|--------------|  
|Number|The absolute value of Number must be greater than 1.|  
  
## Return value

A single decimal value.
