# ACOT

Returns the principal value of the arccotangent, or inverse cotangent of a number.
  
## Syntax  
  
```js
ACOT(number)  
```
  
### Parameters  
  
|Term|Definition|  
|--------|--------------|  
|Number|The cosine of the angle you want. Must be a real number.|  
  
## Return value

A single decimal value.
